Написать программу, которая позволяет задать с помощью консольного ввода несколько геометрических фигур на плосоксти (2D). 
Фигурам присваиватся порядковый номер в соответствии с порядком их ввода. 
После окончания ввода выводятся порядковые номера фигур в порядке убывания их периметра. 
Возможные типы фигур - прямоугольник (задается с помощью ввода ширины и высоты), 
круг (задается с помощью радиуса), треугольник (задается с помощью координат трех его вершин). 
Использовать принципы ООП.
